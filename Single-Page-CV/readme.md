#Semantic HTML CV Project

A structured, single-page professional curriculum vitae built entirely with native, semantic HTML.

The core focus of this project was to master web layout structure, document hierarchy, and accessibility standards without relying on CSS or styling frameworks. Styling and visual layout will be implemented in a subsequent phase.

## Key Features
- Structured using standard HTML5 elements (`<header>`, `<main>`,`<address>`,`<article>` `<section>`, `<footer>`) to ensure optimal machine readability.
- Implemented a clean heading outline (`<h1>` through `<h3>`) to map out career history and educational milestones logically.
- Accessibility (a11y) Baseline: Built with screen-readers in mind
